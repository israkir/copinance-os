# Analytics

Documentation for **derived** market analytics in Copinance OS—starting with **listed options**: Black–Scholes–Merton sensitivities (Greeks) and the **`OptionsChain.metadata`** contract shared by data providers and estimators.

Implementation code lives under **`copinance_os.data`**: vendor adapters in **`data.providers`**, Greek estimation in **`data.analytics.options`**, and the composed options-aware market provider in **`data.providers.market`**. See [Architecture](/developer-guide/architecture) for layer rules.

## In this section

- **[BSM & Greeks (QuantLib)](/analytics/bsm-and-greeks)** — European BSM engine, how to read Δ, Γ, ν, Θ, ρ, and QuantLib’s theta convention.
- **[Options chain metadata](/analytics/options-chain-metadata)** — Reserved `metadata` keys (provider inputs, Greek run provenance), Yahoo implied-vol normalization, env/profile tuning, code layout.
