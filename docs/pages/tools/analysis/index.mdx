# Analysis Tools

Advanced analytical functions for market regime detection, technical analysis, and other research capabilities.

## Available Tools

### Market Regime Detection

Identify bull/bear markets, volatility regimes, and market cycles using rule-based and statistical methods.

- [Market Regime Detection](market-regime) - Comprehensive guide to market regime detection tools

## Overview

Analysis tools provide sophisticated methods for understanding market conditions and making informed investment decisions. These tools are designed to work seamlessly with agentic workflows, allowing AI agents to automatically select and use the appropriate analysis method based on the research question.
