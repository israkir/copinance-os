# Analysis Tools

Advanced analytical functions for market regime detection, technical analysis, and other analysis capabilities.

## Available Tools

### Market Regime Detection

Identify bull/bear markets, volatility regimes, and market cycles using rule-based and statistical methods.

- [Market Regime Detection](market-regime) - Comprehensive guide to market regime detection tools

### Macro Economic Indicators

Comprehensive economic data across rates, credit, commodities, and other categories for macro regime analysis.

- [Macro Economic Indicators](macro-indicators) - FRED-based and proxy indicators, integration with question-driven analysis

## Overview

Analysis tools provide sophisticated methods for understanding market conditions and making informed investment decisions. They work seamlessly with question-driven analysis, allowing the AI to select and use the appropriate analysis method based on your question.
