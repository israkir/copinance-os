# Tools Overview

Copinance OS provides a comprehensive suite of tools for market analysis. Tools are executable functions that can be used in analysis, especially question-driven analysis where the AI can dynamically select and use tools to answer questions.

## Tool Categories

### Analysis Tools
Advanced analytical functions for market regime detection, technical analysis, and macroeconomic analysis.

- **Market Regime Detection**: Identify bull/bear markets, volatility regimes, and market cycles
- **Macro Economic Indicators**: Comprehensive economic data across 9 categories (47+ indicators)
- See [Analysis Tools](/tools/analysis) for details

### Data Provider Tools
Tools that wrap data provider methods, making them accessible to AI agents and analysis runs.

- **Market Data Tools**: Get quotes, historical prices, search instruments, inspect options chains
- **Fundamental Data Tools**: Access financial statements, SEC filings, fundamentals
- See [Overview](/tools/data-providers) for details

## Using Tools

### In question-driven analysis

Tools are automatically available when using question-driven analyze:

```bash
# Question-driven analysis: AI selects and uses tools (e.g. quote, history) to answer
copinance analyze equity AAPL --question "What is the current market trend?"
```

The AI will automatically select and use the appropriate tools to answer your question.

### Programmatic Usage

```python
from copinanceos.infrastructure.tools.analysis import create_rule_based_regime_tools
from copinanceos.infrastructure.tools.data_provider import create_market_data_tools
from copinanceos.infrastructure.data_providers import YFinanceMarketProvider

provider = YFinanceMarketProvider()

# Create tools
regime_tools = create_rule_based_regime_tools(provider)
data_tools = create_market_data_tools(provider)

# Use tools
result = await regime_tools[0].execute(symbol="AAPL")
```

### Tool Registry

Tools can be registered in a central registry:

```python
from copinanceos.infrastructure.tools import ToolRegistry

registry = ToolRegistry()
registry.register_many(regime_tools)
registry.register_many(data_tools)

# Get tool by name
tool = registry.get("detect_market_trend")
```

## Tool Architecture

All tools implement the `Tool` interface, which provides:

- **Schema Definition**: JSON Schema for LLM function calling
- **Parameter Validation**: Automatic validation of inputs
- **Error Handling**: Structured error responses
- **Metadata**: Additional context in results
