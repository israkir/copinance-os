# Tools Overview

**Tools** are typed, schema-backed callables used in analysis—especially **question-driven** flows, where the LLM selects tools to fetch quotes, history, fundamentals, regime signals, and macro series. Deterministic runs (for example `copinance analyze macro`) use the same machinery inside **`core.execution_engine`** and **`core.pipeline.tools`**.

Copinance OS treats tools as **data and computation adapters**, not as free-form reasoning: the model proposes tool calls; implementations return structured results aligned with domain models.

## Analysis tools

Market regime detection and macroeconomic indicators (trend, volatility, cycles, VIX, breadth, sector rotation, 47+ macro indicators). Used automatically in question-driven and macro analysis.

- [Market Regime Detection](analysis/market-regime) — Trend, volatility, cycle detection; VIX, breadth, sector rotation
- [Macro Economic Indicators](analysis/macro-indicators) — Rates, credit, labor, housing, manufacturing, consumer, global, advanced (FRED + yfinance)

## Data provider tools

Tools that wrap provider methods (quotes, history, search, options, fundamentals). Available to the AI in question-driven runs.

- [Data Provider Tools](data-providers) — Market and fundamental tool reference

## Using tools

### In question-driven analysis

Tools are registered automatically when you use question-driven analyze:

```bash
copinance analyze equity AAPL --question "What is the current market trend?"
```

The model selects tools; outputs are grounded in provider and tool results.

### Programmatic usage

```python
from copinance_os.core.pipeline.tools import (
    create_rule_based_regime_tools,
    create_market_data_tools,
    ToolRegistry,
)
from copinance_os.data.providers import YFinanceMarketProvider

provider = YFinanceMarketProvider()
regime_tools = create_rule_based_regime_tools(provider)
data_tools = create_market_data_tools(provider)

result = await regime_tools[0].execute(symbol="AAPL")

registry = ToolRegistry()
registry.register_many(regime_tools)
registry.register_many(data_tools)
tool = registry.get("detect_market_trend")
```

## Tool architecture

All tools implement the **`Tool`** port (`copinance_os.domain.ports.tools`):

- **Schema** — JSON Schema for LLM function calling
- **Validation** — Typed parameters
- **Structured results** — Errors and payloads suitable for prompts and logs
