# Data Provider Tools

Data provider tools wrap provider methods for use in analysis and question-driven runs. They handle caching, validation, and error formatting automatically.

## Market Data Tools

### `get_market_quote`

Get the current quote for a market instrument.

### `get_historical_market_data`

Get historical OHLCV data for an instrument over a date range.

### `search_market_instruments`

Search market instruments by symbol or display name.

### `get_options_chain`

Get an options chain for an underlying instrument, including expirations, calls, puts, and the underlying price.

## Fundamental Data Tools

### `get_equity_fundamentals`

Get comprehensive fundamentals for an equity.

### `get_financial_statements`

Get income statement, balance sheet, or cash flow data for an equity.

### `get_sec_filings`

Get SEC filing metadata (form, dates, accession number, CIK, URL). In question-driven runs, calls are routed to **`EdgarToolsFundamentalProvider`** (edgartools / EDGAR) when configured; results are **cached** with a time-to-live to limit repeat traffic to SEC servers.

### `get_sec_filing_content`

Download filing body for a CIK + accession (full text, HTML, or index URL depending on `document_type`). Served by the same SEC provider as `get_sec_filings`; large bodies may be truncated with metadata; responses are **cached** (longer TTL than filing lists, since a given accession is immutable).

## Creating Tools

```python
from copinance_os.core.pipeline.tools.data_provider import create_market_data_tools
from copinance_os.data.providers import YFinanceMarketProvider

provider = YFinanceMarketProvider()
tools = create_market_data_tools(provider, cache_manager=cache_manager)
```
