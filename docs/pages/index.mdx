# Copinance OS

Open-source market analysis platform and framework: deterministic instrument/market analysis and question-driven AI analysis.

## What You Get

- **Dual analysis modes**: Deterministic pipelines (equity, options, macro) and question-driven analysis (AI uses tools to answer your questions).
- **Adaptive presentation**: Results adapt to financial literacy level (beginner, intermediate, advanced).
- **Extensible architecture**: Clean hexagonal design; plug in custom data providers and analyzers.
- **CLI and library**: Use from the command line or integrate into any Python application.

## Core Concepts

- **Analysis profiles**: Context for runs (literacy level, preferences). Not user accounts—your app owns users; we own analysis context.
- **Deterministic analysis**: Fixed pipeline (quote, history, fundamentals, regime, etc.).
- **Question-driven analysis**: You ask in natural language; the AI uses tools (quote, history, options, macro, etc.) to answer.
