# Copinance OS Documentation

Welcome to Copinance OS - an open-source market analysis platform and framework.

## What is Copinance OS?

Copinance OS is a Python library that helps you perform comprehensive market analysis with:
- **Dual analysis modes** (deterministic instrument/market analysis and question-driven AI analysis)
- **Adaptive presentation** based on financial literacy level
- **Extensible architecture** for custom data providers and analyzers

## Key Concepts

### Analysis Profiles
Analysis profiles provide context for analysis execution (financial literacy level, preferences). This is **NOT** user management - your app handles users, we handle analysis context.

### Analysis Modes
- **Deterministic**: Predefined analysis pipeline (instrument or market) with consistent steps
- **Question-driven**: AI-powered analysis that adapts to natural-language questions
